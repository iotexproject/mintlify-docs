# API Usage

## Q: How do I get an API key?

A: Visit [gateway.iotex.ai/console/token](https://gateway.iotex.ai/console/token) to obtain your exclusive API key. Each customer has an independent key to ensure usage security and accurate statistics.

## Q: Do API keys have an expiration date?

A: API keys are valid long-term by default, but we recommend regular rotation for security reasons. If you need to rotate your key, please contact our technical support.

## Q: Can I use multiple API keys simultaneously?

A: Yes, you can apply for different API keys for different projects or environments, making usage statistics and management easier.
