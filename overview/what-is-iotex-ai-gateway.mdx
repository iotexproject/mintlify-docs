# What is IoTeX AI Gateway

IoTeX AI Gateway provides a unified AI model interface service that allows you to access multiple mainstream AI models including GPT, Claude, Gemini, and more through a single API. No need to register with individual AI service providers separately - one interface handles all your needs.
